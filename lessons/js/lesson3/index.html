Тема: Цикли в JavaScript

Мета уроку:
Навчитися використовувати цикли в JavaScript для автоматизації виконання повторюваних завдань.

  • for цикл

    for цикл є найбільш загальним і широко використовується для повторюваних завдань.
    Синтаксис циклу виглядає так:

                        
    for (let i = 0; i < 5; i++) {
        console.log(i); // Виводить числа від 0 до 4
    }

    У цьому прикладі змінна i ініціалізується значенням 0, цикл виконується до тих пір, поки i менше 5, та збільшує i на 1 кожну ітерацію.

  • while цикл:

    while цикл використовується, коли умова визначається перед початком виконання циклу. Приклад:

                        
    let j = 0;
    while (j < 5) {
        console.log(j); // Виводить числа від 0 до 4
        j++;
    }

    Цикл виконується, поки умова j < 5 виконується. У кожній ітерації j збільшується на 1.

  • do-while цикл:

    do-while цикл гарантує виконання хоча б одного циклу перед перевіркою умови. Приклад:

                        
    let k = 0;
    do {
        console.log(k); // Виводить числа від 0 до 4
        k++;
    } while (k < 5);

    Цей цикл виконується, і потім перевіряє умову k < 5. Якщо умова виконується, цикл повторюється.

  • for ... in цикл:

    перебирання масиву по індексам, а об'єкта по його ключам

                        
    var arr = [
        {
            name: 'Volodia',
            age: 11
        },
        {
            name: 'Maryna',
            age: 14
        }
    ];

    // у звичайному масиві повертає індекси

    for (var el in arr) {
        console.log (el); // 0, 1
    }

    // в об'єкті повертає назви ключів

    for (var el in arr[0]) {
        console.log (el); // name, age
    }

  • for ... of цикл:

    перебирання масиву по значенням елементів :

                        
    var arr = [1, 5, 18, 26];

    for (var el of arr) {
        console.log (el);
    }

  • forEach цикл:

    популярний метод для перебирання масиву з допомогою циклу

                        
    var arr = [1, 5, 18, 26];
    arr.forEach(function (el) {
        console.log(el);
    })

  • Оператори break та continue:

    • Оператор break припиняє виконання циклу. Приклад:

                                  
    for (let m = 0; m < 10; m++) {
        if (m === 5) {
            break; // Зупинка циклу, якщо m дорівнює 5
        }
        console.log(m);
    }

    • Оператор continue переходить до наступної ітерації циклу, пропускаючи код нижче його. Приклад:

                                  
    for (let n = 0; n < 5; n++) {
        if (n === 2) {
            continue; // Пропуск числа 2 і перехід до наступної ітерації
        }
        console.log(n);
    }

    Ці оператори дозволяють керувати виконанням циклів, додаючи умови для виходу з циклу або переходу до наступної ітерації.

Наведені приклади спростять розуміння базових концепцій використання циклів в мові JavaScript. Важливо практикувати та експериментувати з цими структурами для найефективнішого їх використання у реальних програмах.
