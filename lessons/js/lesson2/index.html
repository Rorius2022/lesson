Тема: Типи даних в JavaScript

JavaScript це слабко типізована мова програмування, підтримує різні типи даних.

  • Рядки (Strings)

    Рядки є одним із фундаментальних типів даних у JavaScript і вони використовуються для представлення текстової інформації. У JavaScript рядки можна оголосити за допомогою лапок - одинарних ('), подвійних ("), або обернених (```) лапок. Рядки можуть включати літери, цифри, спеціальні символи та пробіли.

                        
    let messageSingleQuotes = 'Це рядок з одинарними лапками.';
    let messageDoubleQuotes = "Це рядок з подвійними лапками.";
    let messageBackticks = `Це рядок з оберненими лапками та можливістью вставки змінних: ${variable}`;

    Рядки також підтримують різні методи, такі як length (для отримання довжини рядка), charAt (для отримання символу за індексом), concat (для об'єднання рядків) та інші.

  • Числа (Numbers)

    Тип даних "number" в JavaScript використовується для представлення числових значень. Це може бути ціле число або число з плаваючою комою.

                        
    let integerNumber = 42;
    let floatingPointNumber = 3.14;

  • Булеві значення (Booleans)

    Булеві значення в JavaScript можуть бути лише true або false. Вони використовуються для умовних виразів і логічних операцій.

                        
    let isTrue = true;
    let isFalse = false;

    Булеві значення найчастіше використовуються в умовних виразах та логічних операціях, таких як && (логічне "І"), || (логічне "АБО") та ! (логічне "НІ").

  • Масиви (Arrays)

    Масиви в JavaScript - це упорядковані колекції значень, кожен з яких може бути іншого типу даних.

                        
    let fruits = ['яблуко', 'груша', 'апельсин'];
    let numbers = [1, 2, 3, 4, 5];

    Масиви в JavaScript починаються з індексу 0, і до них можна звертатися та модифікувати за допомогою різноманітних методів, таких як push, pop, shift, unshift та інші.

  • Об'єкти (Objects)

    Об'єкти в JavaScript - це структури даних, які дозволяють зберігати дані у форматі ключ-значення.

    Приклад:

                        
    let person = {
        name: 'John',
        age: 30,
        isStudent: false
    };

    Об'єкти можуть містити різні типи даних, включаючи рядки, числа, масиви, інші об'єкти, функції та більше. Звертатися до значень в об'єкті можна за допомогою ключів.

  • Undefined та Null

    Undefined в JavaScript вказує на те, що значення не було призначено змінній.

    Null використовується для вказання на відсутність значення чи пустоту.

    Приклад:

                        
    let undefinedVariable;
    let nullValue = null;

    Ці типи даних досить специфічні, і undefined зазвичай виникає, коли змінній не присвоєно значення, тоді як null може бути присвоєно в якості відсутнього значення.

Вивчення цих типів даних дозволяє розробникам створювати потужні та гнучкі програми в JavaScript, використовуючи різноманітні структури даних та оператори. Вони є важливою частиною будь-якого програміста, який працює з мовою JavaScript.
