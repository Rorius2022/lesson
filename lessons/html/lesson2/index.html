Урок 2: Текстові елементи HTML

Заголовки та абзаци

Теги <h1> до <h6> використовуються для створення заголовків різного рівня. <h1> - найбільший, <h6> - найменший. Тег <p> визначає абзаци тексту.

Форматування тексту

Теги <strong>, <em>, та <u> використовуються для форматування тексту: надання жирності, курсиву та підкреслення відповідно.

Списки

Теги <ul>, <ol>, та <li> використовуються для створення списків. Тег <dl> використовується для списку термінів, а <dt> та <dd> для визначення термінів та їх значень.

Вирівнювання тексту

Теги <div> та <span> використовуються для вирівнювання тексту, групування елементів та створення контейнерів. Теги <br> та <hr> додають розрив рядка та горизонтальну лінію відповідно.

Цитати та посилання

Теги <blockquote> та <q> використовуються для виділення цитат. Тег <cite> вказує на ім'я автора цитати. Тег <a> використовується для створення гіперпосилань.

Спеціальний текст

Теги <code> та <pre> використовуються для відображення тексту з фіксованою шириною шрифта та збереження форматування тексту (коду) відповідно.

Супер- та підскрипти

Теги <sup> та <sub> використовуються для визначення суперскрипту та підскрипту відповідно.
