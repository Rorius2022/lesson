Урок 3: Списки в HTML

Ненумеровані списки (`<ul>`)

Ненумеровані списки використовуються для представлення елементів без порядкового номеру.

            
                <ul>
                    <li>Перший елемент</li>
                    <li>Другий елемент</li>
                    <li>Третій елемент</li>
                </ul>

Нумеровані списки (`<ol>`)

Нумеровані списки використовуються для представлення елементів з порядковим номером.

            
                <ol>
                    <li>Перший елемент</li>
                    <li>Другий елемент</li>
                    <li>Третій елемент</li>
                </ol>

Вкладені списки

Можливо вкладати один вид списку всередину іншого, створюючи багаторівневі структури списків.

            
                <ul>
                    <li>Елемент 1</li>
                    <li>Елемент 2
                        <ul>
                            <li>Вкладений елемент 1</li>
                            <li>Вкладений елемент 2</li>
                        </ul>
                    </li>
                    <li>Елемент 3</li>
                </ul>

Списки термінів (`<dl>`, `<dt>`, `<dd>`)

Список термінів використовується для представлення термінів та їхніх визначень.

            
                <dl>
                    <dt>HTML</dt>
                    <dd>Мова розмітки для створення веб-сторінок.</dd>

                    <dt>CSS</dt>
                    <dd>Мова стилізації для визначення вигляду веб-документів.</dd>
                </dl>

Змінення маркера (`type`, `start`, `reversed`)

Атрибут `type` визначає тип маркера для нумерованих списків (`1`, `A`, `a`, `I`, `i`).

Атрибут `start` встановлює початкове значення для нумерованого списку.

Атрибут `reversed` змінює порядок нумерації на зворотний.

            
                <ol type="A">
                    <li>Перший елемент</li>
                    <li>Другий елемент</li>
                </ol>

                <ol start="3">
                    <li>Третій елемент</li>
                    <li>Четвертий елемент</li>
                </ol>

                <ol reversed>
                    <li>Третій елемент</li>
                    <li>Другий елемент</li>
                    <li>Перший елемент</li>
                </ol>

Списки та посилання

Можливо використовувати списки для створення навігаційних меню та інших структур.

            
                <ul>
                    <li><a href="#">Головна</a></li>
                    <li><a href="#">Про нас</a></li>
                    <li><a href="#">Контакти</a></li>
                </ul>
