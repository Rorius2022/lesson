Урок 1: Основи HTML

Вступ до HTML

HTML (Hypertext Markup Language) - це стандартна мова розмітки, яка використовується для створення веб-сторінок. HTML дозволяє розміщувати та структурувати інформацію на веб-сторінці. Він визначає елементи сторінки та їхню взаємодію.

HTML використовує "теги" - елементи у кутових дужках, які визначають різні частини вмісту, такі як заголовки, абзаци, посилання тощо. Теги формують структуру сторінки та надають їй семантичність.

Структура HTML-документу

HTML-документ починається із тегу <!DOCTYPE html>, який вказує версію HTML. Основна структура включає теги <html>, <head>, і <body>.

<html> - це кореневий елемент, який обгортає всі вміст сторінки. <head> містить мета-інформацію, таку як заголовок сторінки <title>. <body> містить основний зміст сторінки.

            
                <!DOCTYPE html>
                <html>
                <head>
                    <title>Назва вашої сторінки</title>
                </head>
                <body>
                    <!-- Зміст сторінки тут -->
                </body>
                </html>

Основні теги

HTML використовує різні теги для визначення елементів сторінки. Наприклад, теги <h1>...<h6> використовуються для визначення заголовків різного рівня, де <h1> - найвищий рівень, а <h6> - найнижчий.

Тег <p> використовується для визначення абзаців тексту. Теги <strong>, <em>, <u> використовуються для форматування тексту, роблячи його жирним, курсивним та підкресленим відповідно.

            
                <body>
                    <h1>Ласкаво просимо на наш сайт!</h1>
                    <p>Це перший абзац нашого тексту. <strong>Важлива інформація</strong> може бути виділена жирним шрифтом.</p>
                    <p>Також можна використовувати <em>курсив</em> для підкреслення тексту.</p>
                </body>

Робота з текстовим вмістом

HTML дозволяє вставляти звичайний текст та використовувати теги для форматування та стилізації тексту. Ми можемо додавати списки, таблиці та інші елементи для впорядкування інформації на сторінці.

Наприклад, теги <ul>, <ol>, та <li> використовуються для створення ненумерованих та нумерованих списків.

            
                <body>
                    <h2>Робота з текстовим вмістом</h2>
                    <p>HTML дозволяє вставляти звичайний текст та використовувати теги для форматування та стилізації тексту.</p>
                    
                    <h3>Списки</h3>
                    <ul>
                        <li>Ненумерований пункт списку</li>
                        <li>Ще один ненумерований пункт списку</li>
                    </ul>

                    <ol>
                        <li>Нумерований пункт списку</li>
                        <li>Інший нумерований пункт списку</li>
                    </ol>
                </body>
