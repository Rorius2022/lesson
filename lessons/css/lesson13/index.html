Тема: CSS transform

CSS transform - це потужний інструмент для створення анімацій та зміни стилів об'єктів на сторінці. Використання CSS transform дозволяє створювати різні типи трансформацій, такі як зміна масштабу, поворот, зсув, перекос, тощо.

  • scale() - зміна масштабу

                        
    .box.scale {
        transform: scale(1.5); /* Збільшимо масштаб в 1.5 рази */
    }

    Подивитися приклад реалізації

  • rotate() - поворот.

                        
    .box.rotate {
        transform: rotate(45deg); /* Повернемо на 45 градусів */
    }

    Подивитися приклад реалізації

  • translate() - зсув.

                        
    .box.translate {
        transform: translate(50px, 20px); /* Зсунемо вправо на 50px та вниз на 20px */
    }

    Подивитися приклад реалізації

  • skew() - перекос.

                        
    .box.skew {
        transform: skew(20deg, 10deg); /* Перекосимо на 20 градусів по осі X та 10 градусів по осі Y */
    }
