Тема: CSS селектори nth-*

CSS селектори nth-child, nth-of-type і nth-last-child дозволяють вибирати елементи на сторінці за їх позицією в батьківському елементі.

  • nth-child дозволяє вибирати елементи за їх позицією в батьківському елементі. Він використовується з функціональною нотацією an + b

    • a - це коефіцієнт, який визначає періодичність. Він вказує, як часто повторюються відповідні елементи.
    • n - це спеціальна змінна, яка представляє позицію елемента в батьківському контейнері.
    • b - це зсув, який вказує, з якого елемента починати вибір.

    Приклади функціональної нотації:

    • 2n вибирає кожен другий елемент (парні).
    • 3n вибирає кожен третій елемент.
    • 4n + 1 вибирає кожен четвертий елемент, починаючи з першого елемента.
    • n вибирає кожний елемент (кожну позицію).

    Функціональна нотація дозволяє точно контролювати, які елементи вибирати на основі їх позиції в батьківському контейнері. Вона корисна для створення різних стилів та розмітки, особливо для списків та таблиць, де потрібно обирати конкретні рядки або колонки.

                        
    /* Вибір кожного другого елемента списку (парні) */
    li:nth-child(2n) {
        background-color: #e74c3c;
    }
    
    /* Вибір третього елемента списку */
    li:nth-child(3) {
        background-color: #27ae60;
    }

    Подивитися приклад реалізації

  • nth-of-type дуже схожий на nth-child, але він вибирає елементи за їх позицією, яка специфікується для конкретного типу елементу. В інших словах, він враховує лише типи елементів, а не всі елементи в батьківському контейнері.

                        
    /* Вибір другого елемента li в списку */
    li:nth-of-type(2) {
        background-color: #e74c3c;
    }

    Подивитися приклад реалізації

  • nth-last-child вибирає елементи за їх позицією, враховуючи, що вони рахуються від кінця батьківського контейнера. Він може бути корисним для вибору останніх елементів списку.

                        
    /* Вибір останнього елемента списку */
    li:nth-last-child(1) {
        background-color: #27ae60;
    }

    Подивитися приклад реалізації

Ці селектори дозволяють точно керувати тими елементами, які ви бажаєте стилізувати, з урахуванням їх позиції в батьківському контейнері. Вони можуть бути корисними для створення стилів для конкретних елементів списку, таблиць та інших структур на сторінці.


Деякі приклади для кращого розуміння функціональної нотації:
  • Відображає непарні рядки HTML таблиці: 1, 3, 5, тощо. 
    
    tr:nth-child(odd) або tr:nth-child(2n+1)
  • Відображає парні рядки HTML таблиці: 2, 4, 6, тощо. 
    
    tr:nth-child(even) або tr:nth-child(2n)
  • Відображає сьомий елемент. 
    
    :nth-child(7)
  • Відображає елементи 5 [=5×1], 10 [=5×2], 15 [=5×3], тощо. Перший результат формули - 0 [=5x0], що не відповідає жодному елементу, оскільки елементи індексуються з 1, а n починається з 0. Це може здатися дивним на перший погляд, але стає зрозумілим, коли B-частина формули >0, як у наступному прикладі. 
    
    :nth-child(5n)
  • Відображає сьомий і всі наступні елементи: 7 [=0+7], 8 [=1+7], 9 [=2+7], тощо. 
    
    :nth-child(n+7)
  • Відображає елементи 4 [=(3×0)+4], 7 [=(3×1)+4], 10 [=(3×2)+4], 13 [=(3×3)+4], тощо. 
    
    :nth-child(3n+4)
  • Відображає перші три елементи. [=-0+3, -1+3, -2+3] 
    
    :nth-child(-n+3)
  • Відображає кожен елемент <p> у групі одноосібників. Це вибирає ті ж елементи, що і простий селектор p (хоча з більшою специфічністю). 
    
    p:nth-child(n)
  • Відображає кожен <p>, який є першим елементом у групі одноосібників. Це так само, як селектор :first-child (і має ту саму специфічність). 
    
    p:nth-child(1) або p:nth-child(0n+1)
  • Відображає восьмий по п'ятнадцятий елемент <p> у групі одноосібників. 
    
    p:nth-child(n+8):nth-child(-n+15)
