Тема: CSS Media Queries

CSS Media Queries - це потужний інструмент для створення адаптивних веб-сторінок, який дозволяє пристосовувати вигляд та структуру сторінки до різних пристроїв і розмірів вікна браузера. Ви можете змінювати стилі для різних екранів, планшетів та смартфонів, щоб забезпечити оптимальний користувацький досвід.

Основні поняття:

  • Медіа-запит - це логічний вираз, який визначає, які стилі мають бути застосовані до сторінки в залежності від характеристик пристрою або вікна браузера.

                        
    @media screen and (max-width: 768px) {
        /* CSS стилі для екранів, які менше 768px */
    }

    Подивитися приклад реалізації

  • Параметри медіа-запиту

    • screen: Вказує тип пристрою, для якого застосовується запит. В даному випадку, це екрани.

    • (max-width: 768px): Визначає умову для застосування стилів. В цьому прикладі, стилі будуть застосовані, якщо ширина екрану менша або рівна 768 пікселям.

Загальні поради:

  • Використовуйте медіа-запити для адаптації макету і шрифтів для різних пристроїв.
  • Вимірюйте та тестуйте різні розміри екранів, щоб забезпечити правильну роботу медіа-запитів.
  • Розробляйте сторінку мобільної версії спочатку, а потім застосовуйте медіа-запити для більших екранів.
  • Використовуйте адаптивні картинки та відео, щоб зменшити завантаження сторінки на мобільних пристроях.
