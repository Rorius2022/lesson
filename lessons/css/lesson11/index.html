Тема: position

position в CSS визначає, як елемент має розташовуватися відносно інших елементів на веб-сторінці.

Основні властивості:

  • static (за замовчуванням): Елемент розташовується в потоці документа, і його позиція визначається потоком контенту. Властивості top, right, bottom, left і z-index не мають впливу на елементи зі значенням position: static.

                        
    .static-element {
        position: static;
        /* Елемент розташовується в потоці контенту, і значення top, right, bottom, left не мають впливу. */
    }

    Подивитися приклад реалізації

  • relative: Елемент розташовується відносно своєї поточної позиції за допомогою властивостей top, right, bottom і left. Він все ще займає простір в документі, навіть якщо його позиція змінюється.

                        
    .relative-element {
        position: relative;
        top: 20px; /* Зміщення вниз на 20 пікселів відносно поточної позиції. */
        left: 10px; /* Зміщення вліво на 10 пікселів відносно поточної позиції. */
    }

    Подивитися приклад реалізації

  • absolute: Елемент розташовується відносно ближчого батьківського елемента з position іншим від static, або відносно вікна браузера, якщо батьківський елемент не має такої позиції. Елемент виймається з потоку контенту і може накладатися на інші елементи.

                        
    .parent {
        position: relative;
    }
        
    .absolute-element {
        position: absolute;
        top: 0; /* Розташовується відносно верхнього краю батьківського елемента. */
        right: 0; /* Розташовується відносно правого краю батьківського елемента. */
    }

    Подивитися приклад реалізації

  • fixed: Елемент розташовується відносно вікна браузера і не залежить від прокрутки сторінки. Він також виймається з потоку контенту і може накладатися на інші елементи.

                        
    .fixed-element {
        position: fixed;
        top: 20px; /* Розташовується відносно верхнього краю вікна браузера. */
        right: 20px; /* Розташовується відносно правого краю вікна браузера. */
    }

    Подивитися приклад реалізації

  • sticky: Елемент розташовується відносно батьківського контейнера з прокруткою або вікна браузера, якщо батьківський контейнер зі специфічною висотою зміщується поза видиму область. Він діє як relative, поки не досягає вказаної точки, після чого стає fixed.

                        
    .sticky-container {
        position: relative;
        height: 300px;
        overflow: auto; /* Для створення прокрутки в контейнері. */
    }
        
    .sticky-element {
        position: sticky;
        top: 10px; /* Почне залишатися на місці, коли досягне 10 пікселів від верхнього краю контейнера. */
    }

    Подивитися приклад реалізації

Властивості top, right, bottom, left

Ці властивості використовуються для зміщення елемента відносно його поточної позиції. Наприклад, top: 10px; зміщує елемент вниз на 10 пікселів відносно його поточної позиції.

Властивість z-index

Властивість z-index визначає порядок накладання елементів один на одного. Елементи з більшим значенням z-index накладуться на ті, у яких воно менше.

Використання властивості position

position корисно для створення адаптивних та складних макетів. Наприклад, вона дозволяє створювати прикріплені навігаційні панелі, виносні вікна, та інші інтерактивні компоненти.

Поширені сценарії використання

position: relative часто використовується для створення анімацій та зміщення елементів у відповідь на події користувача.

position: absolute і position: fixed корисні для створення навігаційних меню, виносних вікон, а також для закріплення елементів на сторінці.

position: sticky зазвичай використовується для створення прикріплених заголовків таблиць та меню, які накладаються на зміст сторінки при прокручуванні.
