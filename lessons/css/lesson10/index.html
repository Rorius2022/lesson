Тема: display: flex;

display: flex - це CSS властивість, яка встановлює контейнеру гнучкий контекст для розміщення дитячих елементів в одному ряду або колонці.

Основні властивості:

  • display: flex; - встановлює контейнеру режим flexbox.

                        
    .container {
        display: flex;
    }

    Подивитися приклад реалізації

  • flex-direction - визначає напрямок розміщення елементів (рядок або колонка).

                        
    .container {
        display: flex;
        flex-direction: row; /* або column */
    }

    Подивитися приклад реалізації

  • flex-wrap - вказує, чи повинні переважати елементи на одному ряду чи обгортатися на наступний рядок (або колонку).

                        
    .container {
        display: flex;
        flex-wrap: wrap; /* або nowrap, wrap-reverse */
    }

    Подивитися приклад реалізації

  • justify-content - вирівнювання елементів по горизонталі.

                        
    .container {
        display: flex;
        justify-content: start; /* або center, end, space-between, space-around */
    }

    Подивитися приклад реалізації

  • align-items - вирівнювання елементів по вертикалі.

                        
    .container {
        display: flex;
        align-items: center; /* або flex-start, flex-end, baseline, stretch */
    }

    Подивитися приклад реалізації

  • align-content - вирівнювання контейнера по вертикалі, якщо є декілька рядків (для flex-wrap: wrap;).

                        
    .container {
        display: flex;
        flex-wrap: wrap;
        align-content: center; /* або flex-start, flex-end, space-between, space-around, stretch */
    }

    Подивитися приклад реалізації

  • order - це CSS властивість, яка визначає порядок відображення елементів flexbox в контейнері.
    За замовчуванням всі елементи мають значення order: 0;, і вони відображаються в порядку, в якому вони з'являються в дереві DOM.
    Змінюючи значення order, ви можете змінити порядок відображення елементів. 

                        
    .item {
        order: 1; /* Змінює порядок відображення */
    }

    Подивитися приклад реалізації

  • flex-grow - це CSS властивість, яка встановлює розтягнення елементів flexbox у відношенні один до одного в контейнері.
    Значення flex-grow вказує, наскільки елемент має розширюватися відносно інших елементів flexbox зі значенням flex-grow менше за 1.
    За замовчуванням всі елементи flexbox мають значення flex-grow: 0;, що означає, що вони не розширюються. 

                        
    .item {
        flex-grow: 1; /* Розширюється відносно інших елементів flexbox */
    }

    Подивитися приклад реалізації
