Тема: CSS units (одиниць виміру)

У Cascading Style Sheets (CSS) існує безліч різних одиниць виміру, які дозволяють встановлювати розміри та розташування елементів на веб-сторінці. Кожна одиниця виміру має свої особливості та призначена для різних завдань. Ось кілька з найпопулярніших одиниць виміру CSS:

Абсолютні одиниці виміру

  • Пікселі (px): це найменша одиниця виміру на екрані. Один піксель, зазвичай, відповідає одному фізичному пікселю на моніторі. Ця одиниця зазвичай використовується для точного контролю розмірів елементів.

                        
    div {
        width: 200px;
        height: 100px;
    }

    Подивитися приклад реалізації

  • PT і PC: PT (points) та PC (picas) - це одиниці виміру, які використовуються для друку та, зазвичай, не використовуються в веб-розробці. PT зазвичай відповідає 1/72 дюйма, а PC - 12 точкам.

                        
    p {
        font-size: 12pt; /* Розмір шрифту встановлений в 12 точок */
    }                                                    
                    
                
                        
    p {
        font-size: 1pc; /* Розмір шрифту встановлений в 1 піку (1/6 дюйма) */
    }

    Подивитися приклад реалізації

  • CM і MM: CM (centimeters) та MM (millimeters) використовуються для встановлення розмірів в сантиметрах і міліметрах відповідно. Вони рідко використовуються в веб-розробці через різницю в розмірах екранів.

                        
    div {
        width: 5cm; /* Ширина елемента встановлена на 5 сантиметрів */
        height: 2cm; /* Висота елемента встановлена на 2 сантиметри */
    }                                                    
                    
                
                        
    p {
        margin: 10mm; /* Відступи встановлені на 10 міліметрів */
    }

    Подивитися приклад реалізації

  • IN (inches): Дюйми рідко використовуються в веб-розробці, але іноді можуть бути корисні для встановлення розмірів у друкарській продукції.

                        
    div {
        width: 3in; /* Ширина елемента встановлена на 3 дюйми */
    }

    Подивитися приклад реалізації

Відносні одиниці виміру

  • Відсотки (%): дозволяють вказати розмір відносно розміру батьківського елемента. Наприклад, 50% ширини означає половину ширини батьківського елемента.

                        
    div {
        width: 50%;
        height: 75%;
    }

    Подивитися приклад реалізації

  • EM (em): це відносна одиниця виміру, яка базується на поточному розмірі шрифту. Наприклад, якщо розмір шрифту встановлений на 16 пікселів, то 1em дорівнює 16 пікселям.

                        
    p {
        font-size: 1.2em; /* Збільшує розмір шрифту на 20% */
    }

    Подивитися приклад реалізації

  • REM (rem): це одиниця виміру, яка також залежить від розміру шрифту, але на відміну від EM, вона базується на розмірі шрифту батьківського елемента. Це робить її більш передбачуваною.

                        
    body {
        font-size: 16px;
    }
        
    div {
        width: 2rem; /* Еквівалентно 32px (16px * 2) */
    }

    Подивитися приклад реалізації

  • VW і VH: VW (viewport width) та VH (viewport height) вимірюють розмір відносно ширини та висоти видимої області браузера відповідно. Наприклад, 1vw дорівнює 1% ширини вікна браузера.

                        
    div {
        width: 50vw; /* Половина ширини вікна браузера */
        height: 25vh; /* Чверть висоти вікна браузера */
    }

    Подивитися приклад реалізації

Формули та обчислення

  • calc(): Функція calc() дозволяє виконувати арифметичні операції з різними одиницями виміру. Це корисно, наприклад, для створення адаптивних дизайнів.

                        
    div {
        width: calc(50% - 20px); /* Ширина - 20 пікселів */
        height: calc(100vh - 50px); /* Висота - 50 пікселів */
    }

    Подивитися приклад реалізації
