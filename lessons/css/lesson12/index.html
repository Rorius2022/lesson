Тема: Псевдоелементи в CSS

Псевдоелементи - це спеціальні селектори в CSS, які дозволяють вибирати та стилізувати частину елементів, що знаходяться в межах інших елементів, але не являються окремими елементами в дереві DOM.

  • ::before - створює віртуальний псевдоелемент, який вставляється перед вмістом вказаного елемента.

                        
    .element::before {
        content: "Додати текст перед елементом";
    }

    Подивитися приклад реалізації

  • ::after - створює віртуальний псевдоелемент, який вставляється після вмісту вказаного елемента.

                        
    .element::after {
        content: "Додати текст після елементу";
    }

    Подивитися приклад реалізації

  • ::first-line - вибирає перший рядок тексту вказаного елемента.
    Можна змінювати стиль тексту першого рядка, такий як розмір шрифту, колір тексту, міжрядковий інтервал тощо.

                        
    .element::first-line {
        font-weight: bold;
        color: blue;
    }

    Подивитися приклад реалізації

  • ::first-letter - вибирає першу букву тексту вказаного елемента.
    Зазвичай використовується для стилізації великої початкової букви в абзацах або заголовках.

                        
    .element::first-letter {
        font-size: 150%;
        color: red;
    }

    Подивитися приклад реалізації

  • ::selection - дозволяє стилізувати виділену частину тексту, яку користувач виділив мишею або клавішами.

                        
    ::selection {
        background-color: yellow;
        color: black;
    }

    Подивитися приклад реалізації

Властивість content

Властивість content використовується для визначення вмісту псевдоелементів, таких як ::before та ::after.

Вона може приймати значення у вигляді тексту, URL, або спеціальних літералів.

Висновок

Псевдоелементи в CSS дозволяють створювати різноманітні декоративні та структурні елементи на веб-сторінці, які можуть покращити дизайн і функціональність сторінки.

Вони корисні для додавання розділювальних ліній, іконок, відступів і виділених елементів.
