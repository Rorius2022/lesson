Тема: Основні команди в Git

  • Ініціалізація (створення) нового репозиторію.

    Знаходячись в директорії проекту відкрийте консоль git та введіть команду:

    git init

  • git add - додає ваші зміни у файлі до робочої області

    git add - додати певний файл

    git add . - додати всі файли

  • git commit - зберігає зміни у вашому локальному сховищі

    Щоразу, коли ви фіксуєте зміни коду, ви повинні включити короткий опис внесених змін. Це повідомлення про фіксацію допомагає іншим зрозуміти внесені зміни.

    git commit -m “”

  • git push - надсилає зафіксовані зміни файлу з локального репозиторію у віддалений репозиторій, щоб інші могли їх використовувати

    Він також створить іменовану гілку у віддаленому сховищі, якщо вона не існує.

    git push origin
